O que é Client?

Voltar

Os clients são normalmente utilizados na computação para descrever um programa, um dispositivo ou um sistema de solicitação de serviços ou informações de outro programa, conhecido como servidor. O client é responsável por fazer a solicitação da informação que vai para o servidor, e então o servidor retorna para o client, a informação que foi pedida.

Em outras palavras, é como se um cliente chegasse a um restaurante e fizesse ao garçom um pedido, o garçom iria anotar e após algum tempo, retornaria com o pedido que foi feito.

O que são os Navegadores?

Voltar

Os navegadores são softwares gratuitos desenvolvidos para serem usados na busca de informações na internet.

No navegador é possível realizar diversas atividades, dentes elas está a pesquisa de informações, assistir vídeos, tudo que possa imaginar.

Assim que um usuário pesquisa algum conteúdo no navagador, ele interpreta e exibe informações das páginas WEB que foram escritas em HTML, CSS e JavaScript.

HTML (HyperText Markup Language), CSS (Cascading Style Sheets) e JavaScript

Aplicações WEB

Voltar

As aplicações WEB são softwares ou programas que são acessados através de navegadores WEB. Também conhecidos como programas que funcionam na internet e não precisam ser instalados pelo usuário para serem utilizados.

As aplicações WEB também são desenvolvidas utilizando tecnologias como HTML, CSS e JavaScript no lado client e PHP, Python, Ruby, entre outras, pelo lado servidor.

Como foi falado, temos as aplicações WEB que são através de softwares ou programas que não precisam ser instalados e quando se instala o aplicativo, se tornam não aplicações WEB.
